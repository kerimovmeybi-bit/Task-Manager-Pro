# Task Manager Pro

## Описание

Приложение для управления задачами с возможностью создания, удаления и изменения статуса.

## Стек

- React
- TypeScript
- Redux Toolkit
- React Router
- Material UI
- JSON Server

## Функционал

- CRUD задач
- Фильтрация
- Асинхронные запросы
- Страница задачи
- Навигация

## Запуск проекта

### frontend

npm install
npm run dev

### backend

npx json-server db.json --port 3001
